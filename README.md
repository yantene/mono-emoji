UNCO emoji font
===============

UNCO is a **un**i**co**lored emoji font.
It's based on [Twemoji](https://github.com/twitter/twemoji).

## Features

- This font has no color information (Skin tone modifiers? What?)
- Most emojis are derived from Twemoji
